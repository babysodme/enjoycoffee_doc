# 员工手册

# 前言

本员工手册是依据公司运营的实际情况而制定的，各位员工应必须遵守员工手册中的各项内容，如有需要，公司会就其中内容以通告方式予以修改。如各位员工有任何疑问，可向您你所在的部门主管或行政部负责人咨询。 本员工手册所规定内容由各部门主管监督执行。 本员工手册为本公司员工专用，全体员工须对其中内容保密，不得对外传阅。公司行政部可根据业务及管理的需要，对员工手册进行修订，并拥有解释权。

# 总则

  1. 员工必须遵守公司各项规章制度，严格履行劳动合同。
  2. 服从组织调动，接受领导分配的各项任务，听从指挥，严格遵守操作规程使各项工作符合标准和质量要求。
  3. 严格遵守考勤制度和岗位责任，不迟到、早退、不矿工、不串岗、不擅离职守。
  4. 按规定班次工作，未经批准不得擅自变更、调换班次。
  5. 按规定参加会议和业务学习，不断提高自身的服务技能。

# 职前培训

新员工会受接受到公司安排的职前培训，以了解本公司的文化、公司所要求的行为准则、规章制度和公司动作未来发展的规划。此外，新员工所属任职的部门直属上司还将安排其熟悉工作环境和部门同事，让新员工了解其工作职责。

# 试用期

  1. 新进员工，试用期为 3个月，特殊情况，经总经理核准后另定。
  2. 试用期间或届满时，不符合公司录用标准者，予以退工。
  3. 试用期内的员工不享有特别休假。

# 劳动合同

  1. 试用合格的员工，公司将与其签署劳动合同。
  2. 劳动合同的续订及解除，双方都需要按照员工手册的规定时间提前通知对方。 兼职员工：需提前二周通知 全职员工：需提前一个月通知
  3. 解除、终止、续订劳动合同应当依照《中华人民共和国劳动合同法》和国家及北京市有关规定执行。

# 考勤

  1. 考勤内容：

    • 上班时间已到而未到岗者，即为迟到。
    • 未到下班时间而提前离岗者，即为早退。
    • 工作时间未经领导批准离开工作岗位者，即为擅离职守。
    • 迟到、早退或擅离职守超过30分钟，或未经准假而不到班者，均为旷工。

  2. 考勤管理

    • 每人每月有3次5分钟以内的迟到机会，超出每迟到、早退一次罚款20元。超出20分钟～30分钟以内罚款50元。
    • 凡旷工1日扣发当月100%的绩效工资。连续累计3日旷工除名。一年累计5日旷工扣除年终各项奖金。一年累计7日旷工除名。

  3. 请假制度

    • 员工请假，假期三天内当店店长核准，三天以上由区域经理核准，六天以上由人事部审批。
    • 店长请假，1－3天由区域经理批准，3天以上报人事部批准。
    • 经理级人员，假期一天内由分管副总核准，一天以上由总经理核准。
    • 请假申报在钉钉系统中完成。

  4. 事假

    • 因事必须本身处理者可请事假，每年积计以7天为限。
    • 员工请事假必须于前一天以书面形式提出申请，否则不予请假。
    • 事假待遇规定：
    • 全年事假累计超过5天（含5天）扣除年终奖金的50％； 全年事假累计超过8天（含8天）扣除年终奖金的100％。
    • 若因特殊情况，未能于前一天办理请假手续，应电话向所在部门上级报告，并于事后补办请假手续，否则以旷工论处。

  5. 病假

    • 员工请病假，一律应出具公立医院证明申请病假，病假单须及时交所在部门审核。
    • 员工在工作年度内，可享受三天全薪病假。如员工工作未满一年，则按比例享受。
    • 如员工的有薪病假休完。则按员工在本企业的工作年限，扣除一定比例的工资。
    • 一年内，员工的病假累计超过30天者，扣除年终奖金70％；超过60天者，扣除年终奖金100％。
    • 病假请假医疗费用另行规定。

# 福利

  1. 保险

依照国家及地方政策法规和公司的有关规定，公司为员工按期缴纳养老保险、医疗保险和失业保险。

  1. 法定假日

法定假 元旦 （1天）春节 （3天）劳动节（1天）国庆节（3天） 运营门店法定假日不休息，节假日当天工资的三倍计发，调休假日由部门日后安排调休；如因工作需要无法安排休息的，当天工资按双倍计发放。除法定节假日，因工作需要加班需向部门主管或经理提出申请，准许后方可加班，加班可以发放相应的加班工资或者调休两者取其一。

  1. 特别假日

婚假

每一位适龄的全职员工享有3天的带薪婚假，但须提前出示相关证明，经批准后方可开始假期，婚假应连续休完。

丧假

直系亲属（父母、配偶、子女）不幸辞世时，全职员工享有2天连续丧假；
旁系亲属（配偶之父母、祖父母、兄弟、姐妹）不幸辞世时，全职员工享有1天丧假。

产假

按国家劳动法有关规定, 女性员工享有产假90天，其中产前假15天。

  1. 带薪年假

在企业入职满三年（不含三年）的员工，每四年开始将给予每年7天的带薪休假及2000元的一次性旅游补贴。

  1. 休假期需提前和公司协商，以不耽误工作为前提，得到许可后方可休假；
  2. 服务年资每增加一年，年假增加一天，最长不超过10天；
  3. 带薪年假不能跨年使用；
  4. 休年假须提前两周与公司申请并提交审批手续；
  5. 员工休年假前一年内有以下情况之一者，不给予年假；
    • 全年出勤率低于95%
    • 事、病假累计逾15天者
    • 无故旷工逾2天者

# 晋升及奖惩

晋升

  1. 公司会根据员工的考核成绩、工作能力、操行表现、发展潜力、教育背景、服务年资等作为晋升准则。
  2. 如有职位空缺，公司会优先考虑晋升内部合乎资格的员工。

奖励
公司定期在各部门评选的基础上，选出优秀员工，设立“优秀员工奖”。

  1. 优秀员工要求：

    • 品行端正，业务水平突出，能成为其它员工的榜样；
    • 对公司有突出贡献的人；
    • 对本员工作或跨部门提出良好建议，并被采用者，予以奖励；
    • 其它。

  2. 奖励方式如下：
    奖励分为：口头嘉奖、记功、奖金、有薪休假、升职等。

纪律处分

  1. 口头警告

    • 进出门店重要权限场所拒绝员工的检查。

    • 上下班不打卡或委托他人打卡、代替他人打卡。

    • 上班不佩戴名牌。

    • 上班时仪容仪表不整，如歪带名牌、头发不整、手插口袋、倚靠墙壁、佩戴除婚戒之外饰物、浓妆艳抹、穿拖鞋上班等。

    • 上班迟到、早退。

    • 不按指定的员工通道出入。

    • 上班时打私人电话或私自会客。

    • 上岗前或下班后无故在门店内逗留。

    • 公共区域逗留串岗，使用客人设施。

    • 与客人交谈有不礼貌的行为举止。如在客人面前梳头、打呵欠、伸懒腰、吹口哨、剪指甲等。上班时间吃零食、看报纸、聊天等。下班时间私自穿制服在店内消费。随时吐痰，乱扔果皮、纸屑、烟头等杂物。在客用场所内粗言秽语、高声喧哗、争吵或嬉笑唱歌。工作时服务效率差，工作粗心。在非吸烟区吸烟。

  2. 书面警告

    • 一个月内迟到、早退三次。
    • 工作不认真，纪律松懈，工作时间听、做与工作无关的事情。
    • 对上级有不礼貌言行举止。
    • 不服从上级的合法、合理命令。
    • 与客人发生争执或对客人不礼貌。
    • 工作疏忽使企业财产受到损失。
    • 擅自动用客人使用的物品或器具。
    • 擅自翻动客人物品。
    • 出示假病假条。
    • 未经当店管理人员同意擅自调班、调休。严重违反卖场管理条例。擅离岗位。当班时打瞌睡。泄露企业机密。工作时间喝酒或酒后上班。对可能发生的事情不汇报或隐瞒。违反企业的安全条例与安全管理制度。将客人、同事财物遗失、破坏，却没有及时上报，对调查提供假信息。

  3. 严重警告

    • 一个月内连续旷工三天。
    • 在企业内挑拨打架事件。
    • 未经许可擅自使用卖场客用物品等。
    • 未经许可擅自将企业财产移到别处。
    • 拒不接受上级或有关部门的调查。
    • 损坏客人与企业财产。
    • 对聚众赌博或其它违法行为知情不报，不规劝，互相包庇。
    • 管理不善，造成企业严重损失。
    • 违反安全防火规章，造成事故隐患。
    • 利用公职谋私利。
    • 私自经商，倒买倒卖。向客人索取财物、小费等。偷钱、伪造单据、与客人串谋损害企业利益。传播、收看黄色淫秽书刊、录像等黄色物品。 酗酒、赌博。

    备注：口头警告一次扣50元，书面警告一次扣100元，严重警告扣200元。 以上提及的种种行为仅举例，过失行为不局限于以上所列的范围。

  4. 开除或辞退

    • 当班时间在酒店内饮酒。
    • 在酒店内销售私人物品。
    • 服务态度恶劣，受到客人书面投诉并经调查属实。
    • 偷窃财物，未构成犯罪的。
    • 私换外币。
    • 涂改、假造单据、企业公文，搬弄是非，诽谤他人，影响团结。
    • 在门店范围内打架斗殴。
    • 蓄意破坏门店设备设施。
    • 未经批准，私自兼职。
    • 不按规定保管或使用剧毒、易燃、易爆物品。玩忽职守，违章操作给企业造成经济损失和责任事故。因管理指挥不当，造成客人对企业服务质量问题进行投诉并经调查属实。在企业内外有损害企业形象的言行举止。明知险情却不及时报告，或接到命令后未及时执行。 经常违反企业规定，屡教不改者。 触犯《治安管理条例》及国家任何法律。

# 悦咖啡成功的员工的素质

健康正常：合理安排饮食起居，保持健康而良好的身体。 礼貌和微笑：让礼貌和微笑发自内心，使客人舒畅满意，上司鼓舞，同事破处融洽。记住，如果礼貌是你的“宝剑”的话，那么微笑则是你的“盾牌”。 谦恭：谦恭是一种美德。 清洁：着装整洁，善于修饰，讲卫生。 守时：有时间观念，提前5分钟上班。 兴趣：帮助发展自己的工作潜力。 自我设计：用合理、有序、有计划的方式处理问题。 助人：乐于助人，乐于服务他人，关心同事。 合作精神：具有团队精神，在工作中、同事之间互相照应，为达到共同目标，最大限度地发挥自己的作用。 服从上司：乐于听从和执行上级的决定和命令，即使遇到误会也会“先服从，后上诉”，给上司予应有的尊重。 自律：学会在各种情况下的自我控制。 责任心和可靠性：具有强烈的责任感，视悦咖啡如家，不需监督可独立完成工作，这样终会得到大家的信任。 适应性与灵活性：能解决新的、不可预见的事件，遇事镇定，熟练地运用既定的原则和程序。 领导潜力：能正确理解形势和同事，能够激励和主动帮助同事完成任务，达成工作目标。 良好的知识与技艺：了解悦咖啡，掌握适当行业知识、技艺。 自信心：敢于坚持已见，在挑战中不怕遭受挫折。

# 悦咖啡最讨厌的20种不良习惯

  1. 只顾眼前利益：注重眼前，谋及长远，既是一种策略，更是一种知慧。
