# 开店流程

整洁美观的用餐环境对于我们的顾客起着至关重要的影响 开店准备工作对于后续的营运有着至关重要的影响

开店内容

  • 开启并检查设施、设备运行状况
    依次开启收银机、音响、咖啡机、磨豆机、热水器、沙冰机并观察运行是否正常，同时检查制冰机、蛋糕柜、吧台冷藏冰箱温度是否正常
  • 补货
    检查营运所需物料是否充足，及时补充，对于库存剩余量不足的物料及时向当班主管汇报。
    补货量：加满分配器
    补货频率：低于50%补货，不低于每小时检查一次。
    执行先进先出原则：先上后下、先左后右、先外后里、先零后整
  • 清洁
    对吧台、外场地面及设施设备进行清洁整理，注意死角卫生
  • 巡视
    对外场进行巡视检查（桌椅、灯具、装饰物等是否完好）
  • 迎接顾客
    确保所有设备运转良好
    确保所有物料准备充分
    确保怀着为顾客提供最优质的产品和最优质服务的精神迎接顾客

# 吧台卫生标准

  • 吧台桌面物品物料归位，无杂物污渍，无个人物品。地面干净无杂物垃圾。
  • 蛋糕柜准备充足，柜面无渍、污迹。
  • 备餐柜干净整洁，所有物品均按规定摆放。
  • 自助台无脏痕、污迹，调味瓶、水壶等清洁完好，调味瓶内装调料不少于三分之二，调料不变质，不发霉，不沉淀。
  • 餐具、玻璃器皿清洁完好，消毒严格，无指纹、水渍、油渍、脏痕、污迹，无裂纹、缺口。
  • 果汁机、Blend机，咖啡机，磨豆机等设备干净清洁，无残留汁液，无污渍。
  • 手冲设备，虹吸壶保持干净，无指纹，水渍污迹，灰尘。
  • 各区域抹布，台布，口布，及时清洗，保持干净无异味。
  • 吧台内冰箱，冰机，冰柜外面清洁无水渍污渍。物品摆放整齐有序。
  • 水池及时清理，清洁用具摆设有序，无食物茶叶残渣。
  • 及时清理垃圾桶，保证无裸露垃圾，无异味。垃圾桶外面无污渍水渍。

# 闭店流程

每天的打烊工作会对第二天的营运起到至关重要的影响,打烊结束后应做到第二天开店即可营业的标准。

  • 准备消毒水
    按照1:29的配比进行配制（半桶水加4瓶盖，抹布浸泡20分）

  • 吧台：
    擦拭所有酱料瓶、盒表面，放入冰箱保存。 清洁台面、设备表面，原物料按照先进先出原则归位。 清洗餐具并按定位码放

  • 清洁咖啡机：
    刷洗冲煮头，反冲、放水、清洗接水盘并擦拭机器表面
    点击查看设备的维护与保养

  • 餐桌
    将客人留下的所有垃圾丢入垃圾桶内，用消毒抹布将桌子椅子擦拭干净，不要将食物残渣擦到地上（重点清洁玻璃桌面、沙发缝隙）
  • 垃圾桶
    经常检查外围垃圾桶，视需要更换垃圾袋
  • 地面
    先清扫地面，以扫除丢弃物，食物及残渣。当扫地时，对顾客要有礼貌，不可要求客人移位。在拖地之前，彻底擦掉所有油渍或打翻的食物。
  • 拖地
    提前备好“小心地滑”的警示牌，并阻挡正在清洁的区域。绝对不允许用不洁的墩布清洁地面。清洁后用洗涤剂彻底清洁拖布，冲洗至清水为止，并加入消毒液、金纺浸泡。
